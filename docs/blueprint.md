# **App Name**: ChronoTask

## Core Features:

- Task Creation: Allow users to create new tasks with titles, descriptions, due dates, and priority levels.
- Task Listing: Display tasks in a clear, sortable list, categorized by due date and priority.
- Due Date and Time Setting: Enable users to specify due dates and times for tasks, including setting up reminders.
- Reminder Notifications: Send timely reminders to users based on their task due dates and times. AI-powered tool that learns optimal notification times based on user behavior.
- Task Completion: Allow users to mark tasks as complete, moving them to a completed tasks list.

## Style Guidelines:

- Primary color: Muted blue (#6699CC) for a calm and focused user experience.
- Background color: Light gray (#F0F0F0) to ensure readability and reduce eye strain.
- Accent color: Soft green (#8FBC8F) to indicate task completion and positive feedback.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean, modern look.
- Use minimalist, consistent icons to represent task status and priority levels.
- Clean, card-based layout for tasks, making it easy to scan and manage items.
- Subtle transitions and animations for task additions, updates, and completions to provide visual feedback.
